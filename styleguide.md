Curative Intelligence Agent - App Style Guide
This document outlines the complete UX/UI style guide for the Curative Intelligence Agent application. It translates the brand's elegant, minimalist aesthetic into a functional design system for a high-touch, intuitive, and scalable SaaS product.

Color Palette
The palette is derived from the brand's warm, neutral tones, enhanced with functional, high-contrast colors for usability and accessibility.

Primary Colors
Brand Tan: #D2B193 (Used for primary buttons, primary accents, highlights, and illustrative backgrounds)
Brand Cream: #EFE8D8 (Used for primary app backgrounds, selected states, content containers, and clean surfaces)

Secondary Colors
Alabaster White: #FBFAF8 (Used for secondary app backgrounds, secondary accents, highlights, and illustrative backgrounds)
Dark Umber: #3A2F2F (Used for primary text and high-emphasis elements)

Accent Colors
Muted Gold: #C1A37E (For key calls-to-action and notifications)
Pale Sage: #D0C9B8 (For decorative elements or inactive states)

Functional Colors
Success Green: #43A047 (For success states and confirmations)
Error Red: #D9534F (For error messages and destructive actions)
Warning Yellow: #FFC107 (For non-critical warnings)
Text Secondary: #7A6F6F (For secondary text, labels, and disabled states)

Background Colors
Surface White: #FFFFFF (Pure white for cards and modals to create lift)
App Background: #FBFAF8 (The primary Alabaster White)
Dark Background: #211A1A (For dark mode's primary background)

Typography
The typography pairs an elegant display font with a clean, legible UI font to balance brand personality with usability.

Font Family
Primary Display Font: Floreal (For major headlines and marketing text)
Primary UI Font: Montserrat (For all interface text, including body copy, buttons, and labels)

Font Weights
Regular: 400
Medium: 500
Semibold: 600
Bold: 700

Text Styles
Headings
H1: 32px / 40px, Floreal, Regular, Letter spacing -0.5px
H2: 26px / 32px, Floreal, Regular, Letter spacing -0.2px
H3: 20px / 28px, Montserrat, Semibold

Body Text
Body Large: 17px / 26px, Montserrat, Regular
Body: 16px / 24px, Montserrat, Regular
Body Small: 14px / 20px, Montserrat, Regular

Special Text
Caption: 12px / 16px, Montserrat, Medium
Button Text: 16px / 24px, Montserrat, Medium
Link Text: 16px / 24px, Montserrat, Medium, Color: Dark Umber (#3A2F2F)

Icons
Style: A clean, lightweight, line-art style (e.g., Phosphor Icons).
Primary Icons: 24dp x 24dp
Small Icons: 18dp x 18dp
Interactive Color: Brand Tan (#D2B193)
Inactive/Decorative Color: Pale Sage (#D0C9B8)

Spacing System
Based on an 8dp grid system for rhythmic, consistent layouts.
8dp: Small spacing (internal padding, icon-to-text).
16dp: Default spacing (element margins, list item padding).
24dp: Medium spacing (between distinct components or sections).
32dp: Large spacing (separating major sections on a page).
48dp - 64dp: Extra large spacing (top/bottom padding for screens).

Motion & Animation
Animations should be subtle, fluid, and sophisticated to enhance the user experience.
Standard Transition: 250ms, cubic-bezier(0.4, 0, 0.2, 1) (for fades, slides).
Emphasis Transition: 350ms, cubic-bezier(0.4, 0, 0.2, 1) (for modals and major state changes).
Microinteractions: 150ms, Ease-out (for hover effects, button presses).
Page Transitions: 400ms, subtle fade-in/fade-out.
